DCAPA - Non-Custodial Parent – Details

Non-Custodial Parent Information

·         Special Circumstances:  Select the circumstance type.

·         Is the Non-Custodial Parent last name unknown?:  Select Yes or No.

·         First:Enter the non-custodial parent’s first name.

·         Middle:  Enter the non-custodial parent’s middle name.

·         Last: Enter the non-custodial parent’s last name.

·         Suffix:  Select the non-custodial parent’s suffix.

·         SSN:  Enter the non-custodial parent’s Social Security Number if enabled.

·         Gender:  Select Male or Female.

·         Race:  Select the race(s).

·         Date of Birth:  Enter the birth date.

·         City of Birth:  Enter the city of birth.

·         Birth State/Province:  Select the birth state/province.

·         Birth Country:  Select the birth country.

·         If deceased, date of death:  Select the date of death.

·         Deceased Verification:  Select the deceased verification, if deceased.

·         City of Death:  Enter the city of death, if deceased.

·         Death State/Province:  Select the death state/province, if deceased.

·         Death Country:  Select the death country, if deceased.

Other Name(s) Information

·         First Name:  Enter the other first name of the non-custodial parent.

·         Middle Name:  Enter the other middle name of the non-custodial parent.

·         Last Name:  Enter the other last name of the non-custodial parent.

·         Suffix:  Select the other suffix of the non-custodial parent.

Father & Mother of the Non-Custodial Parent

Father:

·         First:  Enter the non-custodial parent father’sfirst name.

·         Middle:  Enter the non-custodial parent father’smiddle name.

·         Last:  Enter the non-custodial parent father’slast name.

·         Suffix:  Select the non-custodial parent father’ssuffix.

Mother:

·         First:  Enter the non-custodial parent mother’s first name.

·         Middle:  Enter the non-custodial parent mother’s middle name.

·         Last:  Enter the non-custodial parent mother’s last name.

·         Suffix:  Select the non-custodial parent mother’s suffix.

DCAPC - Non-Custodial Parent – Address Details

Non-Custodial Parent Information

·         Non-Custodial Parent Name:  Displays the non-custodial parent’s name, age and gender.

·         Non-Custodial Parent SSN:  Displays the non-custodial parent’s Social Security Number.

Address

·         Address Line 1:  Enter the non-custodial parent’s address.

·         Address Line 2:  Enter additional address information such as such as P.O. Box number, if necessary.

·         City:  Enter the city.

·         State:  Select the state.

·         Zip/Postal Code:  Enter the zip or postal code. Use both fields if known.

·         In Care Of:  Enter the person’s name for this address if the applicant is using that person and their address to receive mail instead of their own address.

·         Phone Number:  Enter the phone number.

·         Last Known Address Date:  Enter the last known address date.

DCNCE - Non-Custodial Parent – Employment Details

Non-Custodial Parent Information

·         Non-Custodial Parent Name:  Displays the non-custodial parent’s name, age and gender.

·         Non-Custodial Parent SSN:  Displays the non-custodial parent’s Social Security Number.

Employment Information

·         Employer:  Enter the name of the non-custodial parent’s employer.

·         Occupation:  Enter the occupation.

·         Start Date:  Enter the start date.

·         End Date:  Enter the end date.

Employer Address

·         Employer:  Displays the employer’s name from the Employment Information table above.

·         Address Line 1:  Enter the employer’s address.

·         Address Line 2:  Enter additional address information such as P.O. Box number, if necessary.

·         City:  Enter the city name.

·         State/Province:  Select the state/province from the list.

·         Zip/Postal Code 1:  Enter the zip or postal code.

·         Zip/Postal Code 2:  Enter the zip or postal code extension if known.

DCASA - Non-Custodial Parent – Child Support Order

Non-Custodial Parent Information

·         Non-Custodial Parent Name:  Displays the non-custodial parent’s name, age and gender.

·         Non-Custodial Parent SSN:  Displays the non-custodial parent’s Social Security Number.

Financial Support Liability

·         Has child support been ordered?:  Select Yes or No.

·         Date Court Order Effective:  Select the date if the above field is Yes.

·         Was the Custodial parent ever married to this Non-Custodial parent?:  Select Yes or No.

·         Date of Marriage:  Select the date of marriage.

·         Ordered Amount:  Enter the ordered amount.

·         Pay Frequency:  Select the pay frequency.

·         Date of Last Payment:  Select the date of last payment.

·         Court City:  Enter the court city name.

·         Court County:  Enter the court county name.

·         Court State/Province:  Select the state/province.

·         Order/Decree Docket Number:  Enter the order/decree docket number.

·         Where are support payments being made to?:  Select where support payments are made.

·         City Where Support Payments are Made to:  Enter the city name where support payments are made.

·         County Where Support Payments are Made to:  Enter the county name where support payments are made.

·         State/Province Where Support Payments are Made to:  Select the state/province where support payments are made.

·         Child Support Conversion Date:  Displays the conversion date after it is determined by EDBC.

DCAPB - Non-Custodial Parent – Child Connection

Child Connection Information

·         Non-Custodial Parent Name:  Displays the non-custodial parent’s name, age and gender.

·         Non-Custodial Parent SSN:  Displays the non-custodial parent’s Social Security Number.

·         Special Circumstances:  Displays the special circumstances.

Child Details

·         Child Name:  Select the child’s name.

·         Custodial Person Name:  Displays the non-custodial person’s name after you select the child’s name.

·         Absence Reason:  Select the absence reason.

·         Paternity Established:  Select the established paternity.

·         Special Circumstances Start Date:  Enter the special circumstances start date.

·         Special Circumstances End Date:  Enter the special circumstances end date.

DCAPS - Non-Custodial Parent – Summary

·         Non-Custodial Parent Name:  Displays the non-custodial parent’s full name, age and gender.

·         Child Name:  Displays the child associated with the non-custodial parent.

·         Paternity Established:  Displays the paternity established.

·         Special Circumstances:  Displays the special circumstances.

·         Child Active in the Case?:Displays Yes or No based on the household status of the child.

DCAND - Non-Custodial Parent Compliance – Details

Non-Custodial Parent Information

·         Custodial Person:  Select the custodial person’s name if this is a new record.  Otherwise, displays the custodial person’s name.

·         Non-Custodial Parent Name:Displays the non-custodial parent’s name who is associated to the case.

Non-Custodial Parent Compliance Dates

·         Circumstances Start/Change Date(CS/CD):Enter the start or change date for a new circumstance. Displays the first of the month, three months prior to the Application File Date for Intake case action.

·         End Date:  Enter the end date for the situation.

Compliance Information

·         Cooperation Status of Custodial Person:  Select the cooperation status.

DCANS - Non-Custodial Parent Compliance – Summary

·         Custodial Person:  Displays the custodial person’s full name, age and gender.

·         Non-Custodial Parent:  Displays the non-custodial parent’s full name, age and gender.

·         Cooperation Status:  Displays the cooperation status.

·         Circumstances Start/Change Date (CS/CD):Displays the start or change date of the circumstance.
