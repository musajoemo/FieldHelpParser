FWJOB - Batch Exception Report

Summary Search Criteria

·         Begin Date:  Enter the start date for the search.

·         End Date:  Enter the stop date for the search.

·         Job ID:  Enter the ID text of the job (such as AL-FWTO-MLY).

·         Description:  Enter the text about the job (such as, SSA or Corrections).

·         Status:  Select the status of the job (Aborted, Ended, or Started).

·         Job ID:  Displays the ID text of a job.

·         Description:  Displays the description of a job.

·         As Of Date:  Displays the date each program uses for its processing.

·         Start Time:  Displays when the job began running.

·         End Time:  Displays when the job stopped running.

·         Execution Time:  Displays the execution time of the job in HH:MM:SS (hours:minutes:seconds).

·         Status:  Displays the status of the job. If the status is Ended or Aborted, the status is displayed as a hyperlink. Click the hyperlink to display job summary details for Ended jobs and exception details for Aborted jobs.
