XABCD - Merge Individuals

This page captures information for linking two Person IDs in the system to one individual.

Multiple Person ID Information

·         Request Type:Select the merger request type.

Correct Person IX

·         Xagnifying Lens:  Click this button to open the Search Criteria pop-up window to search for the Person ID.

·         Name:Displays the Correct Person’s name.

·         Person ID: Displays the unique ID number assigned to the Correct Person.

XAlias Person ID

·         Magnifying Lens:  Click this button to open the Search Criteria pop-up window to search for the Person ID.

·         Name:XDisplays the Alias Person’s name.

·         Person ID: Displays the unique ID number assigned to the Alias Person.

XDCALT - Protective Payee – Details

This page captures protective payee information for a TANF Assistance Group.

 

XPrimary Person Information

·         Payee:Displays the name of the individual who is paying the expense selected on the Issuance – Details page.

·         Issuance Method:Displays the issuance method code based on the issuance method selected on the Issuance – Details page.

·         Type of Assistance:Displays the type of assistance based on the type of assistance selected on the Issuance – Details page. .

XProtective Payee Details

·         First:  Enter the protective payee’s first name.

·         Middle:  Enter the protective payee’s middle name.

·         Last: Enter the protective payee’s last name.

·         Suffix:  Select the protective payee’s suffix.

·         Phone Number:  Enter the protective payee’s 10-digit phone number.

·         Date of Birth:Enter the protective payee’s date of birth.

·         SSN:  Enter the protective payee’s social security number.

·         Mother’s Maiden Name: Enter the protective payee’s mother’s maiden name.

XProtective Payee Mailing Address

·         Address Line 1:  Enter the protective payee’s address, such as street number and name, P.O. Box or rural route.

·         Address Line 2:Enter any additional address information such as apartment number if necessary.

·         City:  Enter the city name.

·         State: Select a state if other than Indiana.

·         Zip Code:Enter the zip code or postal code. Use both fields if known.

DCALP - Protective Payee – Summary

This page displays a summary of protective payee information.You can also navigate to the page to add or edit the protective payee details.

The following fields display in the table:

·         Case #:  Displays the case number.

·         Type of Assistance:  Displays the type of assistance.

·         Payee:  Displays the name of the payee.

·         Protective Payee:  Displays the name of the protective payee.

·         Address:Displays the individual’s address.
