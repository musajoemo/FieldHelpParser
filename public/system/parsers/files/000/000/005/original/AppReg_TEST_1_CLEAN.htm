ARRAP - Register Application

Application Information

·         Date and Time Registration Initiated: Displays completed with the date and time the application was initiated.

·         Application File Date:  Enter the date the application was received. Displays the application file date if the application is from an electronic interface.

·         Application Signed:  Select Yes or No.

·         Source: Select how the application was received.

·         Did a Navigator help with this application?:  Select Yes or No. This does not apply to Med Enrollment.

·         Was this application submitted by an Authorized Enrollment Center?:Select Yes or No.

Special Applications

·         Is this an application for a special program?:  Select Yes or No.

·         For which program is this an application?:  Select Disaster SNAP, Med Enrollment or Resource Assessment if the above field is Yes.

Initial Contact Person

·         First:  Enter the initial contact person’s first name.

·         Middle:  Enter the initial contact person’s middle name.

·         Last:Enter the initial contact person’s last name.

·         Suffix:  Select the initial contact person’s suffix.

·         Does this individual live in the household?:Select Yes or No.

·         First:  Enterthe navigator’s first name.

·         Middle:  Enter the navigator’s middle name.

·         Last:  Enter the navigator’s last name.

·         Suffix:  Select the navigator’s suffix.

·         ID Number:  Enter the navigator’s ID number.

·         Organization ID Number:  Enter the navigator’s organization ID number.

·         Organization Name:  Enter the navigator’s organization name.

Initial Contact Person – Out of Household Mailing Address

·         Address Line 1:  Enter the initial contact person’s address, such as street number and name, P.O. Box or rural route.

·         Address Line 2:  Enter any additional address information such as apartment number if necessary.

·         City:Enter the city name.

·         State: Select a state if other than Indiana.

·         Zip/Postal Code: Enter the zip code or postal code. Use both fields if known.

·         In Care Of:  Enter the person’s name for this address if the applicant is using that person and their address to receive mail instead of their own address.

ARRAD - Register Application – Address

·         Applicant is Homeless:  Select this checkbox if the applicant is homeless.
