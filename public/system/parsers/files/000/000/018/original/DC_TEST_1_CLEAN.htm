DCRQU - Resource Questions

·         Does any household member own any resources?:  Select Yes or No.

Liquid Assets

·         Does anyone in the household own liquid assets such as cash-on-hand, stocks, bonds, checking or savings accounts, nursing home accounts, promissory notes, annuities, certificates of deposit, or retirement accounts?:  Select No, Pend or Yes.

Trust

·         Does anyone in the household own a trust?:  Select No, Pend or Yes.

Real Property/Contract for Deed

·         Does anyone in the household own any real property/contract for deed such as a home, mobile home/trailer, life estate, contract for deed?:  Select No, Pend or Yes.

Life Insurance

·         Does anyone in the household have life insurance?:  Select No, Pend or Yes.

Prepaid Funeral Agreement / Burial Plot

·         Does anyone in the household own a prepaid funeral agreement/burial plot?:  Select No, Pend or Yes.

Other Resource

·         Does anyone in the household own any other resource such as farm or business equipment or items of unusual value?:  Select No, Pend or Yes.

Livestock

·         Does anyone in the household own livestock?:  Select No, Pend or Yes.

Oil and Mineral Rights

·         Does anyone in the household own any oil and/or mineral rights?:  Select No, Pend or Yes.

Vehicle

·         Does anyone in the household own a vehicle such as a car, truck, motorcycle, camper, trailer, snowmobile, motor home, all-terrain vehicle, watercraft/boat or aircraft?:  Select No, Pend or Yes.

Resource Transfer 36/60 Months

·         Has anyone given away or loaned money or property; sold or transferred real property or other resources within the last 36 months (or 60 months if a trust)?:  Select No, Pend or Yes.

DCRLQ - Liquid Asset – Details

Person Information

·         Name:Select the individual’s name if this is a new record.  Otherwise, displays the individual’s name.

·         Person ID:  Displays the unique ID number assigned to an individual. This field populates after the Name field is selected.

Liquid Asset Dates

·         Circumstances Start/Change Date:  (CS/CD) Enter the start or change date for a new circumstance. Displays the first of the month, three months prior to the Application File Date for Intake case action.

·         End Date:  This field is disabled and will display the final resource date if one is entered on this page.

·         Reported On:  Enter the date the client reported a change. Displays the Application File Date for Intake case action.

·         Date Client Became Aware: Enter the date the client became aware of the change. Displays the Application File Datefor the Intake case action.

·         Verification Received On: Enter the date the client provided verification.

Liquid Asset Details

·         Liquid Asset Type:  Select the liquid asset type.

·         Financial Institution:  Enter the financial institution except if the above field is Cash.

·         Account Number:  Enter the account number if enabled.

·         Does the resource earn interest or dividends?:  Select Yes or No if enabled.

·         Is Liquid Asset jointly owned?:  Select Yes or No except if the Liquid Asset Type field is a Retirement Account.

·         Was this asset placed into a trust?:  Select Yes or No.

·         Verification:  Select the verification.

·         End Reason:  Enter the end reason except for historical records.

·         Final Resource Date:  Enter the final resource date if an end reason is entered for the above field.

Liquid Asset Amount Summary

·         Gross Value:  Enter the gross value amount.

·         Verification:  Select the verification for the above field.

·         Penalty Percentage for Early Withdrawal:  Enter the penalty percentage for early withdrawal if enabled.

·         Amount Owed:  Enter the amount owed if enabled.

·         Verification:  Select the verification if an amount is entered for the above field.

·         Net Value:  Displays the system-calculated net value which is the gross value minus the calculated penalty percentage and/or the amount owed. Displays 0 if the gross value is not provided or if the calculated penalty percentage and/or the amount owed is greater than the gross value.

Transfer

·         Was Liquid Resource transferred?:  Select Yes or No.

Availability

·         Available?:  Select Yes or No.

·         Availability Verification:  Select the verification if the above field is No.

DCRCD - Liquid Asset – Additional Details for Annuity and Promissory Note

Person Information

·         Name:Displays the individual’s full name, age and gender.

·         Person ID:  Displays the unique ID number assigned to the individual on the case.

Dates

·         Circumstances Start/Change Date:  (CS/CD) Enter the start or change date for a new circumstance. Displays the CS/CD from the Liquid Assets – Details page.

·         End Date:  Enter the end date for the situation, if applicable.

Annuity

·         Annuity Payment being received by:Select the individual(s) in the household who are receiving the annuity payment, including if there are others outside of the household. 

·         Characteristic:Select the characteristic.

·         Financial Institution:Enter the financial institution name.

·         Date Purchased/Converted:Enter the date the annuity was purchased or converted.

·         Fair Market Value:Enter the fair market value amount.

·         Annuity Verification:Select the annuity verification.

Promissory Note

·         Characteristic:Select the characteristic.

·         Financial Institution:Enter the financial institution name.

·         Promissory Note Principal Balance:Enter the promissory note principal balance amount.

·         Date Payment Received:Enter the date payment was received.

·         Rebutted Value:Enter the rebutted value amount.

·         Lien Amount on Promissory Note:Enter the lien amount on the promissory note.

·         Equity:  Displays the equity value calculated as rebutted value minus lien amount on promissory note. If rebutted value is not available, but promissory note principal balance is available, equity value is calculated as promissory note principal balance minus lien amount.

·         Is this Promissory Note negotiable?:  Select Yes or No.

·         Is this Promissory Note secured:Select Yes or No.

·         Secured By:Enter who secured the promissory note.

·         Secured Value:Enter the secured value amount.

·         Verification:Select the verification if enabled.
