RLCRL - Create Reception Log Entry – Reception Log Entry

IndividualInformation

·         First:  Enter the individual’s first name.

·         Middle:  Enter the individual’s middle name.

·         Last:  Enter the individual’s last name.

·         Date of Birth:  Enter the individual’s date of birth.

·         SSN:Enter the individual’s Social Security Number.

·         Person ID:Enter the unique ID number assigned to the individual.

·         Case or Application # (Dropdown): Select “Case” or “Application” to enter a Case or Application #.

·         Case/Application # (Textbox): This field is enabled when a value is selected in the previous field. If “Case” is selected in the previous field, enter the individual’s case. If “Application” is selected in the previous field, enter the individual’s application number.

Representative Information

·         Relationship to Individual:  Select the representative’s relationship to the individual.

·         First:  Enter the representative’s first name for all relationships except for Self.

·         Middle:  Enter the representative’s middle name for all relationships except for Self.

·         Last:  Enter the representative’s last name for all relationships except for Self.

·         Suffix:  Select the representative’s suffix for all relationships except for Self.

·         Method of Contact:  Select the method used to make contact.

·         Phone Number:Enter the individual’s or representative’s phone number.

RLCPU - Create Reception Log Entry – Purpose

Reception Log Entry Purpose Information

·         Purpose:  Select the purpose for creating the reception log entry.

·         Program:  Select the associated program(s) if enabled.

·         Verification(s):Select the verification(s) presented by the individual if enabled.

·         Application Form(s):  Click the ‘APPLICATION FOR SNAP AND CASH ASSISTANCE (STATE FORM 53263)’ hyperlink to open the PDF form.

·         Other Purpose Description:  Enter a purpose description if Purpose is Other.

Purpose Summary

·         Purpose:  Displays the purpose.

·         Detailed Purpose:  Displays the detailed purpose.

·         Does the individual want to register to vote?:  Select Yes or Client Declined VR Form.

·         Actions Taken:  Enter a description of the actions taken regarding this log entry.
