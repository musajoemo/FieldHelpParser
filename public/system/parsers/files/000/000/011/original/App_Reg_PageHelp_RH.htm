AMMAS - Maintain Application – Summary

This page displays an overall summary of the application, including all individuals on the application, and the programs for which each individual applied. This is the final page of the Application Registration process where you submit and complete the application.

 

 If you want to edit any program information on the application before submitting it, you will need to navigate to the Program Summary page to edit that information. Once you submit the application, you cannot make any changes to the application in Application Registration. For additional informationopen **“How To topic” describing the navigation of this process.**

 If an application is received from an electronic external sourcesuch as the FSSA Benefits Portal, Federally Facilitated Marketplace (FFM), Social Security Administration (SSA) or processed through the document center, the information on this page defaults to the selections on that application and some fields are protected. You will still need to compare the information from the source document with what displays on the page to verify it is correct.

Application Summary

·         Current Primary Individual: Displays the primary individual’s full name, age and gender.

·         Expedited: Displays the result of expedited SNAP screening, if applicable. Otherwise it displays N/A (Not Applicable).

·         Application Signed:Displays Yes or No to indicate if the application is signed.

·         Program:Displays all programs selected for each individual on the application.

·         Other Name(s):Displays any other legal name(s) for each individual on the application.
